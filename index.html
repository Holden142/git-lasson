• Клонирование репозитория (в Git Bash): git clone ***путь***
  • git status – статус текущего репозитория
  • git add ***название файла*** - добавить файл под контроль гита
  • git commit -am "описания изменения " – зафиксировать изменения и дать им коментарий
  • git push – отправить изменения на сервер
  • git pull – забрать изменения
